About This Interactive Mockup

This browseable mockup demonstrates the comprehensive vision for Practera's AI-first educator workspace. Each page showcases different features that capture learning signals across meetings, documents, code repositories, and mixed reality environments.

Note: This is a functional prototype with simulated data. All AI interactions, student data, and system integrations are designed to demonstrate the intended user experience.

Key Innovation Areas:

  • Off-System Learning Capture: AI bots join meetings, monitor documents, track code changes
  • Skills-as-Trajectory Feedback: Real-time learning velocity tracking and intervention
  • Conversational Project Scaffolding: Natural language project setup with AI task decomposition
  • Mixed Reality Ready: xAPI streaming from Unity & Unreal scenarios
  • Automated Intervention Engine: Context-aware nudges and educator empowerment

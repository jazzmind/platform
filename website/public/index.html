About This Mockup

This browseable mockup demonstrates the vision outlined in the Practera Educator Workspace 2.0 document. Each page showcases different AI-powered features that capture learning wherever it happens, guide projects, and prove skill growth in real-time.

Key Features Demonstrated:

  • AI Meeting Capture: Bots join meetings to extract decisions, blockers, and learning moments
  • Document & Code Watchers: Monitor GitHub, Google Drive, SharePoint for learning signals
  • Project Brief Builder: Conversational wizard turns ideas into SMART goals
  • Task Decomposition AI: Auto-creates Kanban tasks with owners and due dates
  • Learning Velocity Analytics: Skills-as-trajectory feedback with real-time tracking
  • Mixed Reality Integration: xAPI streaming from Unity & Unreal scenarios
  • Cross-Modal Dashboards: Combine MR scores with expert ratings and meeting sentiment
